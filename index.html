Primera página web

La primera página que hice fue en la prepa, fue una página en donde colocamos los nombres de los alumnos del salón, con el objetivo de que al seleccionar el nombre de algún alumno, tengamos acceso a los avisos generales, calendario de exámenes, etc.

Más programación

Mi proyecto tuvo enfoque a un emprendimiento de ropa y objetos sublimados, ésto con el objetivo de que el cliente tuviera acceso a los catálogos y precios. También para tener mayor facilidad al momento de hacer algún pedido. Éste proyecto se planea irlo mejorando con varias dinámicas dentro de la página para que el cliente obtenga descuentos y sea más dinámica la página web.

Mi experiencia...

Llevo muy poco programando, es algo que disfruto hacer. Considero que cuento con el conocimiento básico para crear una página web para algún emprendimiento o negocio.
